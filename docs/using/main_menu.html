Главное меню

    Меню Файл

           Новый - создает новый шаблон.
       Открыть - открывает существующий шаблон.
        Сохранить - сохраняет шаблон на диске.
        Сохранить как - сохраняет шаблон под другим именем и в другом формате.
        Параметры страницы - открывает диалог настройки параметров страницы.
        Печать - печать отчета на принтере.
        Просмотр - предварительный просмотр страниц отчета.
        Свойства - открывает диалог для редактирования свойств отчета.
           Выход - закрывает отчет.

    Меню Правка

        Отменить - отменяет предыдущее действие.
        Вернуть - возвращает отмененное действие.
        Вырезать - вырезать выделенный фрагмент отчета в буфер.
        Копировать - копировать выделенный фрагмент отчета в буфер.
        Вставить - вставить фрагмент отчета из буфера.
        Удалить - удаляет содержимое ячеек в выделенном фрагменте отчета.
           Выделить все - выделяет весь лист отчета.
        Поиск - открывает панель поиска внизу отчета.

    Меню Лист

Содержит команды относящиеся к текущему листу отчета.

    Новый - добавляет новый лист к отчету.
    Удалить - удаляет текущий лист отчета.
    Загрузить - загружает отчет из файла и добавляет его как новый лист (листы).
    Сохранить - сохраняет текущий лист в файл как отдельный отчет.
    Влево - перемещает текущий лист влево в списке листов.
    Вправо - перемещает текущий лист вправо в списке листов.
    Печать - печать текущего листа на принтере.

    Меню Формат

        Объединить ячейки - объединяет выделенные ячейки.
        Разрыв строки - устанавливает нижнюю границу страницы по текущей строке.
        Разрыв столбца - уснавливает правую границу страницы по текущему столбцу.

    Меню Дизайнер

           Источник данных - открывает окно для настройки JDBC источников данных.
        Функция - открывает окно со списком пользовательских функций.
        Переменные - открывает окно со списком переменных отчета.
        Свойства группы - открывает окно свойств группы детализации.

           Подменю Группы - вставляет в шаблон отчета соответствующие группы. При удерживании клавиши Shift появляется окно с запросом количества вставляемых строк.
                Заголовок отчета - вставляет в шаблон строки из группы Title.
                Заголовок страницы - вставляет в шаблон строки из группы PageHeader.
                Заголовок группы - вставляет в шаблон строки из группы GroupHeader.
                Детализация - вставляет в шаблон строки из группы Detail.
                Итог группы - вставляет в шаблон строки из группы GroupFooter.
                Простая строка - вставляет в шаблон обычные строки.
                Итог страницы - вставляет в шаблон строки из группы PageFooter.
                Итог отчета - вставляет в шаблон строки из группы Footer.
                Новая группа - вставляет в шаблон новую группу детализации вместе с заголовком и итогом группы.

           Подменю Итоги - вставляет в шаблон отчета итоговые результаты. Если признак вставляется в итог или заголовок группы то суммация происходит по данным группы, если вставляется в итог отчета, то по всему отчету. Вычисляется значение по данным поля или переменной указанной в ячейке, если поле или переменная не указаны, то вычисляется значение по столбцу.
                Сумма - суммирует данные.
                Минимум - находит минимальное значение.
                Максимум - находит максимальное значение.
                Среднее - находит среднее арифметическое значение.

        Не повторять - помечает ячейки как не повторяющие.
        Сформировать - формирует отчет по текущему шаблону.                 

    Меню Вид
   
    Устанавливает внешний вид программы на основе выбранного look&feel.

    Меню Помощь

           Руководство пользователя - открывает этот раздел помощи.
        API Specification - открывает окно помощи с разделом API.
           O JDBReport - показывает диалог с информацией о программе.
