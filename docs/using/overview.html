JDBReport - это приложение для формирования отчетов. Оно может работать как самостоятельное приложение, так и как библиотека для подключения к другим приложениям.
JDBReport написано на java, поэтому оно работает на любой платформе где установлена  Java VM версии 1.5 или выше.
JDBReport позволяет очень быстро, с помощью WYSIWYG редактора, спроектировать и получить нужный отчет. Источниками данных для отчета могут служить запросы из баз данных через JDBC, путем передачи в отчет SQL запросов. Причем к отчету можно подключать несколько источников из разных баз данных. Также в отчет можно передавать java классы, массивы и различные коллекции классов, а также изменять содержимое отчета непосредственно из программы. JDBReport полностью поддерживает язык выражений EL 2.2.
Полученные отчеты могут экспортироваться в различные форматы: HTML, PDF, ODF (ODS, ODT),  Excel 2003, Excel 2007.

JDBReport состоит из генератора отчетов и дизайнера. Генератор используется непостредственно в приложении для генерации отчета вывода его на экран, печать и экспорта в другие форматы. Дизайнер - это визуальный редактор для проектирования шаблона отчета.
