Установка и запуск


    Скачайте с сайта www.jdbreport.com архив jdbreport.tar.gz или jdbreport.zip. Распакуйте его в какую нибудь папку.
Архив содержит библиотеки необходимые для работы JDBReport.

    Для генерации отчетов требуется библиотека jdbreport-rt-2.x.jar, также обязательна библиотека с реализацией интерфейса EL 2.2. Это может быть juel 2.2 или el 2.2..
Опционально могут потребоваться библиотеки:
  • iText для экспорта отчетов в PDF. Поддерживается iText-2.1.7 и iText-5.x. http://www.lowagie.com/iText/
  • POI версии 3.6 или выше, для экспорта в Excel 2003 и Excel 2007 http://poi.apache.org/
  • Batik версии 1.7 или выше, для вставки в отчет графики в формате svg. http://xmlgraphics.apache.org/batik/
  • Для получения данных из баз данных могут потребоваться соответствующие JDBC драйвера.
  • Для отображения стандартных диалогов и сообщений на русском языке понадобится библиотека swing_ru.jar
    Для проектирования шаблонов требуется библиотека jdbreport-dt-2.x.jar. API документация содержится в архиве doc.zip.
Файлы jdbreport-rt-2.x-sources.jar и jdbreport-dt-2.x-sources.jar содержат исходные коды JDBReport.

    Библиотека jdbreport-rt-2.x.jar распространяется по лицензией Lesser General Public License (LGPL) Version 2, jdbreport-dt-2.x.jar - под лицензией GNU General Public License (GPL) Version 2.
  Сторонние библиотеки распространяются под разными лицензиями, для уточнения смотрите документацию производителей.

Для запуска дизайнера отчетов в Linux используется скрипт jdbreport.sh, в Windows jdbreport.exe или jdbreport.cmd.
