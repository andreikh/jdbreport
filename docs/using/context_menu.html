Контекстное меню ячейки

        Свойства Alt+Enter - открывает диалог со свойствами выделенных ячеек.
           Редактор Ctrl+Enter - открывает окно для редактирования содержимого ячейки.
        Объединить ячейки - объединяет выделенные ячейки.
           Подменю Картинка
                Загрузить - загружает картинку из файла.
                Масштабировать - растягивает или сжимает картинку по границам ячейки.
                Удалить - удаляет картинку из ячейки.
                Сохранить - сохраняет картинку в файл в указанном формате.
            Высота строк - открывает диалог для установки высоты строк.
            Ширина столбцов - открывает диалог для установки ширины столбцов.
        Функция - открывает окно со списком пользовательских функций.
        Вырезать - вырезать выделенный фрагмент отчета в буфер.
        Копировать - копировать выделенный фрагмент отчета в буфер.
        Вставить - вставить фрагмент отчета из буфера.
        Удалить - удаляет содержимое ячеек в выделенном фрагменте отчета.
